# Objective
The objective of this project is to analyze airline operational performance by evaluating passenger traffic patterns, carrier-wise efficiency, and route-based flight distribution. The dashboard provides insights into which carriers, locations, and time periods contribute most to overall aviation demand, enabling airlines to optimize capacity planning, enhance passenger engagement, and improve load factor performance.
